# ILI9341

## Language

Spin2, PASM2

## Category

Display

## Description

ILI9341 is a spin2 object that interfaces the ILI9341 chip with the Propeller 2 P2X8C4M64P

Learn more about the Propeller 2 at [www.parallax.com](http://www.parallax.com).
